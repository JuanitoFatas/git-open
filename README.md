# Git::Open

[![Build Status](https://travis-ci.org/JuanitoFatas/git-open.svg?branch=master)][travis]

[travis]: https://travis-ci.org/JuanitoFatas/git-open

## Usage

```
$ git open
```

Open the repository in your browser, yay!

## Installation

```
$ gem install git-open
```
